Work In Progress in the Apache Beam Project

As mentioned in the Contribution Guide, all work in Beam is tracked in our Apache JIRA. In addition, the following types of work may be of particular interest to the Beam community.

  • Starter Tasks
  • Feature Branches

Starter Tasks

The community regular tags good getting started tasks with the label starter. Use a quick JIRA search to identify ways you can get started contributing to Beam.

Feature Branches

Larger features with multiple active developers may be developed on a feature branch before being merged in the master branch. In particular, this is often used for initial development of new components like SDKs or runners.

Current branches include:
